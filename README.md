# Supermarket Sales Analysis Dashboard

This Power BI dashboard provides an in-depth analysis of sales data for a supermarket shop. It offers insights into various key metrics related to sales performance, including profits, total sales, top products, product categories, and more. 
The dashboard includes interactive features such as filters for months, years, payment modes, and sale types, allowing users to customize their analysis based on specific criteria.

## Dashboard Features

### 1. Profit Analysis
- Visualizations displaying profit trends over time, allowing stakeholders to track profitability and identify periods of growth or decline.

### 2. Total Sales Overview
- Analysis of total sales revenue, including trends, distribution by months and years, and comparison with previous periods.

### 3. Filters
- Interactive filters for months and years enable users to focus on specific time periods for detailed analysis.
- Filters for payment modes and sale types allow users to segment sales data based on different transaction attributes.

### 4. Top Products and Product Categories
- Visualizations showcasing the top-selling products and product categories, along with their respective sales volumes and revenues.

## How to Use

1. **Download**: Clone or download the repository to your local machine.

2. **Open**: Open the .pbix file using Power BI Desktop.

3. **Connect Data**: If necessary, connect the dashboard to your sales data source by editing the data connections in Power BI Desktop.

4. **Explore Dashboard**: Utilize the interactive features such as filters for months, years, payment modes, and sale types to customize your analysis.
   
## Data Sources

The dashboard is built using sales data sourced from the internet. The dataset includes information on sales transactions, product details, customer demographics, and transaction attributes such as payment modes and sale types.

## Requirements

- Power BI Desktop (Download from [here](https://powerbi.microsoft.com/desktop/))
- Sales data in a compatible format (e.g., CSV, Excel)
